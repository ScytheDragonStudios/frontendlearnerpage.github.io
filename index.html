Lists

The List tag comes in two parts and has two forms

The first is the 'UL' tag which is Unordered List

    Example
  • Watch Anime
  • Drink Water
  • Code Video Game

The second is 'ol' or Ordered List

    Example
  1. Eat Dinner
  2. Brush Teeth
  3. Sleep

These lists are made possible with the 'li' tag that goes in
both OL and UL.

Tables

Tables are a nifty way of showing data on a website. It use to be
the goto to set up your webpage, but now we use CSS,
Cascading Style Sheets. More on that below.

Playstation Game Plat? Did I Earn?
CyberPunk 2077 Yes No
Sam and Max Yes Yes

The 'Table' tag is first put, then you use 'TR' to set up Rows.

'TD' is then used to put data into the table.

To make the first row bold, you use 'TH' or Table Header, instead
of 'TR'.

To ensure a border appears on your table, in the table tag you
add 'Border'

CSS

Cascading Style Sheets, or, CSS is what's used to stylize our webpages

CSS can change colors, fonts, sizes, animations, and can even do
cool word effects like text glitches or rainbow.

You set up CSS in a couple different ways:

  • External file using a Link rel='stylesheet' href='location'.
  • Inside the body tag within the 'style' tag at the top.
  • Directly in the tag being styled
